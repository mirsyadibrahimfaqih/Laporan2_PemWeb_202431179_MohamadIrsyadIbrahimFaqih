A. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web modern?

CSS digunakan untuk mengatur tampilan halaman agar konsisten, menarik, dan mudah dibaca. JavaScript memberikan interaktivitas pada web seperti animasi, tombol dinamis, dan validasi. Kombinasi keduanya menciptakan web yang modern dan responsif.

B. Jelaskan konsep dan tujuan utama dari Responsive Web Design (RWD)!

Responsive Web Design (RWD) adalah teknik desain agar website menyesuaikan tampilannya dengan ukuran layar perangkat. Tujuannya membuat pengguna nyaman membaca dan berinteraksi di berbagai perangkat, tanpa perlu zoom atau scroll berlebihan.

C. Bagaimana kombinasi CSS, JavaScript, dan prinsip RWD dapat menghasilkan tampilan web yang interaktif sekaligus adaptif?

CSS membuat tampilan rapi, RWD menjadikannya fleksibel di semua ukuran layar, sedangkan JavaScript menambah efek interaktif. Ketiganya saling melengkapi untuk menciptakan pengalaman pengguna yang dinamis dan menarik.
